# Schemat dodania stacji radiowej
Jeśli chcesz dodać stację radiową której nie ma na poniższej liście możesz użyć tego schematu w radios.json:

```js
{
    name: "nazwa stacji radiowej",
    src: "link do strumienia",
    type: "audio/mp3"
}
```
